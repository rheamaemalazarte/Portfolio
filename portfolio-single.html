Website

Electronic Equipment Waste Forecasting System

This is a capstone project that I made in hopes to solve the problem with manually tracking electronic equipment and its waste. It turned out to be a life-changing project that altered my life and made me learn a lot of knowledge.

Research

The project went through a lot of processes before becoming a website. It took a lot of brainstorming and surveys. A qualitative analysis was done in order to support this project.

Wireframing

Wireframing was done in Figma to draw overviews of the project to establish the structure and flow of possible design solutions.

Design

The design was first sketched on paper and later transferred to a designing application for better security and more adequate access to more tools.

Development

The development process was done in Sublime and with the help of XAMPP. Languages used are PHP, HTML, CSS, Javascript, and MYSQL.
